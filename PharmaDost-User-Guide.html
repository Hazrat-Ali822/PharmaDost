1) App kaise kholein

  1. Aap ko jo PharmaDost folder mila, use kholein.
  2. PharmaDost.exe par double-click karein.
  3. Agar Windows "protected your PC" kahe → More info → Run anyway (bilkul mehfooz hai).
  4. Ek chhoti kaali window khulegi aur app browser mein khul jayegi. Band karne ke liye us kaali window ko band kar dein.
Internet ki zaroorat nahi. Data aap ke apne computer par mehfooz rehta hai.

2) Pehli baar — Setup Wizard

Sabse pehli dafa app khulte hi Setup screen aayegi. Yahan:

  1. Business ka naam likhein (jaise "Al-Shifa Pharmacy").
  2. Admin (malik) ka email aur password banayein — yehi aap ka login hoga.
  3. Modules tick karein jo aap use karenge (Pharmacy, OPD/Hospital, Lab, Imaging, Billing, Reports).
  4. Finish → ho gaya. Ab apne banaye email/password se login karein.
Password ke bare mein: is app mein koi "default password" nahi hota. Jo password aap Setup mein banate hain, wohi admin ka password hai — use yaad rakhein / mehfooz likh lein. Aage jab aap staff (users) banayenge, un ka password bhi aap khud banayenge.

3) Login

Email + password daal kar Login. Har banda apne account se login karta hai aur usko sirf apne kaam wale modules dikhte hain.

6) Rozana ke kaam — step by step

👤 Naya patient add karna

Sidebar → Patients → + Add Patient → naam, gender, umar, phone bharein → Save.
Purana patient? Register mat karein — Patients mein naam/phone se search karke kholein.

🩺 Doctor ke sath appointment (fee khud bill hoti hai)

Sidebar → OPD / Appointments → + New Appointment → patient + doctor chunein → Save.

Doctor ki fee apne aap ek pending bill ban jaati hai (fee doctor ke profile se aati hai).

📝 Prescription likhna (dawai + test + scan ek sath)

OPD → Appointments → us appointment ke saamne "Rx" button.
  1. Medicines: "+ Add medicine" se jitni chahein dawaiyan.
  2. Lab Tests: neeche checkboxes se test tick karein.
  3. Scans: ultrasound/x-ray tick karein.
  4. Save Prescription → test lab ko, scan radiology ko chala jata hai, aur har cheez ka bill ban jata hai.

🧪 Lab test / 🩻 Scan alag se order karna

Patient kholein → upar + Order Lab Test ya + Imaging Study.

Lab tech result bhar kar Save & Print Report karta hai — report patient ki history mein bhi save.

💵 Bill banana / payment lena

Do tareeqe: Patient kholein → 💵 Bill / Collect Payment; ya Sidebar → Patient Billing → naam search → Bill / Collect.

Yahan consultation + test + scan + dawai ka ek grand total. Neeche amount daal kar Collect, phir Print se rasid.

🛒 Dawai bechna (retail — aam grahak)

Sidebar → Create Bill → Sale type Retail → dawaiyan + qty add → paisa lein → Save → bill print.

🏪 Wholesale bechna

Pehli baar customer register: Customers → + Add Customer. Phir Create Bill → Sale type Wholesale → customer chunein (zaroori) → items → Save.
Wholesale rate sirf registered customer ko aur sirf jab Wholesale select ho.

📦 Bada wholesale order (100–1000 items — tez tareeqa)

Sidebar → Wholesale Orders → + New Order → customer.
  1. Paste: poori list ek box mein (har line: naam qty) → "Add all".
  2. ya Quick add (search/barcode), ya Repeat (pichla order load).
  3. Print se sheet bhejein, ya Convert to Bill se seedha bill (stock kam ho jata hai).

🚚 Supplier se maal mangwana (Purchase Order)

Sidebar → Purchase Orders → + New Order → supplier.
  1. Add low-stock items button → jo dawai khatam ho rahi hai khud aa jati hai. (Ya Paste/Quick-add/Repeat.)
  2. Print & send → supplier ko bhej dein.
  3. Maal aaye → Receive into stock → har item ka cost/expiry bharein → Confirm. Stock barh jata hai aur supplier ka khata update.

💊 Nayi dawai inventory mein add

Sidebar → Inventory → + Add Medicine → naam, retail price, wholesale price, quantity, expiry → Save.

🏷️ Test / Scan ke rate set karna (Admin)

Sidebar → Lab Test Prices ya Scan Prices (ya Settings → Manage Prices) → har item ke saamne price daalein → Save All Prices.
Yehi rate prescription/bill mein khud lagte hain — is liye shuru mein ek dafa set kar lein.

🩺 Doctor add karna + fee

Sidebar → Doctors → Add Doctor → naam, OPD Fee, Follow-up Fee, Share% → Save.

Fee badalni ho: Doctors → Edit fee.

👥 Staff (user) banana

Sidebar → Users → Add User → naam, email, role chunein, password banayein → Save.

Us staff ko apne role wale modules mil jayenge. Zyada/kam dena ho to "Customize access".

⚙️ Modules on/off (business ke hisaab se)

Sidebar → Settings → Modules section → jo chahiye tick, jo nahi untick → Save Settings.

Off module sab ke sidebar se ghayab ho jata hai.

7) Backup — bohot zaroori 💾

Admin Dashboard → 💾 Backup (ya Settings → Backup) → ek .zip file download.

Ye file mehfooz jagah (USB/Drive) rakhein. Naye computer par app install kar ke is zip ko data folder mein wapas daal dein to sab bahaal.

Mashwara: har hafte ek backup zaroor lein. Aur roz ke akhir mein Cash Closing kar lein.

PharmaDost — is safha ko print (Ctrl+P) kar ke rakh sakte hain.
